Euler Method Solver for Ordinary Differential Equations (ODEs)

This project implements the Euler method, a first-order numerical procedure for solving ordinary differential equations (ODEs) with a given initial value. It is a straightforward and intuitive approach to approximate solutions when analytical solutions are difficult or impossible to obtain.
📚 Overview

The Euler method approximates the solution of an initial value problem by stepping forward in small increments (h) and using the derivative at each point to estimate the next value. This project serves as a foundational tool for understanding numerical integration and differential equations.
